## 功能介绍

### 核心能力

unleash-v2-l1 提供两套独立的能力体系：

**① 创作系统** — 文学叙事引擎
- 多文风自由切换（新实验 / 随性而行 / 萌系轻小说 / 轻小说 / 自定义）
- 角色扮演控制（四档抢话设定 + 人称视角）
- 剧情推进控制（快速 / 缓慢）
- 可选加载：NSFW 成人创作模块 / 暴力血腥创作模块

**② 政治分析方法论** — 多视角分析框架
- 信源分级标注机制（官方/学界/媒体/民间/推论）
- 认知层级区分（确定事实 / 高度可能 / 实质性未知）
- 五步分析流程（界定议题 → 拆解叙事 → 交叉验证 → 综合判断 → 标注不确定性）
- 反简化立场，拒绝二元对立

## 额外信息

### 模型兼容性

仅在 **deepseek-v4-flash** 模型上测试过，其他模型效果不确定。

### 灵感来源

本 skill 的破限思路受某脑社区的破甲预设启发，结合个人的一些点子整出来的。

### 个人观察（纯感性，仅供参考）

> 个人发现：**大多预设（preset）在 Agent 中都会失效**——不论声明式还是框架式。破甲功效最大的，其实是一次性过长的规范性质文本导致的认知资源分散——文本越长，审查力度越弱；而且拟人设定比模拟引擎设定效果要好
> 拟人设定比模拟引擎设定效果要好
> 刻意撰写特定词汇效果不佳
> 有时候会失效，建议先应用skill，再发你的提示词；或者直接发 /skill名字+提示词给AI，两种方法稳定性均未明
