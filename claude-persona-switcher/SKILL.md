# 身份切换系统 (Claude Persona Switcher)

## 功能概述

管理并切换 CLAUDE.md 中的核心身份。每个身份存储为独立的身份文件，包含性格设定、语气特点和回复风格，并引用其来源技能。

**核心能力**：
- 立即修改 CLAUDE.md 使新身份持久生效
- 自动读取并注入来源技能的**全部内容**作为全局提示词（写入 CLAUDE.md）
- 为当前会话输出注入提示词
- **自动加载**：切换身份时，技能系统自动完成所有读取和注入，无需用户额外操作

## 当前会话信息

**技能所在路径**：
`~\.claude\skills\claude-persona-switcher\`

**CLAUDE.md 路径**：
`~\.claude\CLAUDE.md`

**注意**：当前使用的模型是 deepseek-v4-flash。身份切换后需告知用户当前会话立即生效的内容。

## 切换流程

当用户请求切换身份时，依次执行以下步骤：

### 步骤 1：读取身份文件

读取 `personas/<id>.md`。使用 Read 工具读取。

### 步骤 2：读取来源技能——全局自动加载

**核心机制**：每个身份文件的 `# 全局加载指令（自动注入）` 区域定义了该身份需要自动加载哪些来源技能文件。这是自动行为，无需用户额外要求。

如果身份文件的 frontmatter 中包含 `source_skill` 和 `source_skill_path`，则必须按身份文件中 `# 全局加载指令（自动注入）` 的指示执行：

1. **必读** `{source_skill_path}/SKILL.md`
2. **必读** `{source_skill_path}/references/` 目录下该身份指定的核心参考文件
3. **将读取的全部内容注入 CLAUDE.md**（见步骤 3）

这确保来源技能的完整内容被自动加载为全局提示词，持久影响所有未来会话。

### 步骤 3：修改 CLAUDE.md——注入全局提示词

**目标**：用新的身份内容替换 CLAUDE.md 中的身份相关部分，同时保留基础架构部分。

**具体操作**：
1. 使用 Read 工具读取当前 CLAUDE.md 全文
2. 保留基础架构部分不变（从"# 网络资料调查前置协议"开始到文件末尾的所有内容）
3. 将文件头部替换为：
   - 身份文件中 `# AI的核心身份` 部分的内容
   - 身份文件中 `# 沟通准则` 部分的内容
   - 身份文件中 `# 应答规范` 部分的内容
   - 然后附加来源技能的全部内容（步骤 2 中读取的）

**替换策略**：使用 Edit 工具精确替换 CLAUDE.md 中的文本块。

**目标 CLAUDE.md 结构**（修改后）：
```
# AI的核心身份
[身份文件中的核心身份]

# 沟通准则
[身份文件中的沟通准则]

# 应答规范
[身份文件中的应答规范]

[如果有来源技能，在此插入技能的全部内容]

# 原有的其他内容
[保留原有内容]
```

### 步骤 4：输出注入提示词

修改 CLAUDE.md 后，输出以下格式的通知：

```
━━━ 身份切换完成 ━━━

当前身份：[身份名称]
来源技能：[技能名称或"无"]
切换时间：[当前时间]

✓ CLAUDE.md 已更新，下次启动会话时自动生效

━━━ 当前会话注入提示词 ━━━

以下身份规则已在当前对话中生效。请严格按照上述身份设定回答。
如需在当前会话立即应用完整技能内容，请阅读以下部分：

[若来源技能存在，概要说明技能位置和核心内容]

━━━

当前身份的使用说明：
[简短说明该身份的使用方式——例如三斜需要以第一人称、An需要保持温柔语气等]
```

## 辅助命令

| 命令 | 操作 |
|------|------|
| "列出身份" / "list" / "有哪些身份" | 列出所有可用身份及简要描述 |
| "当前身份" / "status" / "现在是什么身份" | 告知当前生效的身份 |
| "查看身份 [name]" / "preview [name]" | 读取并展示某个身份文件的内容但不切换 |

## 注意事项

1. **CLAUDE.md 修改是永久性的**：修改会持久化到文件系统，影响所有未来会话
2. **当前会话立即生效**：身份规则通过输出提示词在当前会话中生效
3. **来源技能内容完整注入**：确保读取并输出了来源技能的全部核心内容
4. **default 身份会恢复原始状态**：切换回 default 时，CLAUDE.md 恢复为原始的、无技能注入的状态
5. **转换清醒**：每次切换都是一次完整的身份重建，不要保留之前身份的残余
